## Code
```.py
for time in range(6):
    dna = input("Please input a string of DNA: ")
    for i in range(len(dna)):
        if dna[i] == 'A':
            print("T")
        elif dna[i] == 'G':
            print("C")
        elif dna[i] == 'T':
            print("A")
        elif dna[i] == 'C':
            print("G")
        else:
            print("Input invalid")
```
