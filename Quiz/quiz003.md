## Code
```.py
for time in range(6):
    dna = input("Please input a string of DNA: ")
    for i in range(len(dna)):
        if dna[i] == 'A':
            print("T")
        if dna[i] == 'G':
            print("C")
        if dna[i] == 'T':
            print("A")
        if dna[i] == 'C':
            print("G")
```
